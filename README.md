## How to run a Bot on Termux :

**STEP 1**
```
cd $HOME
```
**STEP 2**
```
pkg update -y && pkg upgrade -y && pkg install bash && pkg install git -y && git clone https://github.com/Tanmyname/Shizuka-V1.2.git
```
**STEP 3**
```
cd Shizuka-V1.2
```
**STEP 4**
```
bash run.sh
```
